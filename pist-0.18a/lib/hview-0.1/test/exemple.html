Modification de l'aspect du texte

En HTML, il existe énormément de tag permettant de changer facilement l'aspect du texte. On peut facilement mettre en gras ou en italique. Il existe d'autre tag fonctionnel comme strong ou cite,

La fonte de base est medium-roman.
HTML permet d'altérer la fonte de base. Les 3 altérations primaires sont :

  • gras:
    Voici un phrase qui va changer d'aspect.
  • italic:
    Voici un phrase qui va changer d'aspect.
  • mono-spaced font:
    Voici un phrase qui va changer d'aspect.

Il y a 2 autres modifications d'aspect qui peuvent être appréciées dans certain cas :

  • underline:
    Voici un phrase qui va changer d'aspect.
  • strike-out:
    Voici un phrase qui va changer d'aspect.

On peut même pousser la magie en mélangeant les modifications d'aspect ! Par exemple : du gras italique ou une fonte "monospace" soulignée et en gras.

Les tags fonctionnel tags sont :

  • cite - pour les citations
  • code - pour les fragments de codes
  • em - pour l'emphase
  • kbd - pour user input
  • samp - pour les exemples
  • strong
  • var - pour des références é des variables

item		quantity		price

          Tabs within preformatted objects each take up half of an indent level. You can also use line breaks inside lists.
